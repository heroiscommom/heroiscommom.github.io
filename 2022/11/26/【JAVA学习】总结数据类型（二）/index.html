|阅读量:

java数据类型

Java数据类型可以分为整形，浮点型，布尔型，java数据类型分类图：

整型

关于整型分别有：

  • int 在计算机中所占内存空间为4个字节 （32bit）
    范围：-2的31次方–>2的31次方-1；
  • short 在计算机中所占内存空间为2个字节（16bit）
    范围：-2的15次方–>2的15次方-1，即-32768~32767；
  • long 在计算机中所占内存空间为8个字节（64bit）
    范围：-2的63次方–>2的63次方-1；

浮点型

关于浮点型分别有：

  • float在计算机所占内存空间为4个字节（32bit）
    范围：-2的128次方–>-2的149次方
    2的-149次方–>2的128次方，且不包括0；
    保证输出小数点精确到后小数点六位。
  • double在计算机所占内存空间为8个字节（64bit）
    范围：-2的1024次方–>-2的(-1022-52)次方
    2的-1074次方–>2的1024次方，且不包括0；
    保证输出小数点精确到后小数点十五位。

字符型

char在计算机中所占内存空间为2个字节 （16bit）可以表示一个汉字

布尔类型

boolean在计算机中所占内存空间为1个字节。
