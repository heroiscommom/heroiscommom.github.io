【java学习】jdk的解释和path变量的设置（一）

发表于2022-11-26|更新于2022-11-27
|阅读量:

JDK的解释

JDK 是整个 Java 开发的核心，它包括 Java 运行环境、Java 基础类库和 Java 工具。

JDK文件

下载jdk后，bin文件夹会包含javac.exe和java.exe

javac.exe会把java源文件编译为class文件，而java.exe会把class文件运行。

java源代码的运行

而要使用这些运行程序则要在命令窗口实现，而在命令窗口实现代码的编译和运行，则要一层一层的将该文件夹打开，找到如此俩个应用程序（像我的文件夹如此乱，要找到猴年马月），于是在设置path变量后，就直接可以在命令窗口进行编译和运行。

path变量的配置


首先在我的电脑打开属性

选择高级系统设置后，在环境变量中新建

变量名可以写为

JAVA_HOME

变量值可以浏览目录选择你所装JDK那个文件夹

在系统设置选择path变量后，新建一个新的变量

%JAVA_HOME%\bin

然后将其上移到顶端，如此path变量就设置完成了

如此便可以直接在命令窗口编译和运行程序了。
